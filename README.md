# Sang Saka NFT Collection Website

A modern, responsive website showcasing the Sang Saka NFT collection. Built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- Modern, responsive design with a red theme
- Smooth animations and transitions
- Mobile-friendly navigation
- Integration with OpenSea collection
- Featured NFTs showcase
- About section

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Development

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

```bash
npm run build
# or
yarn build
```

Then start the production server:

```bash
npm start
# or
yarn start
```
